Research @ Tryal.AI

This demo is designed to show Tryal.AI's new UI library designed for collecting student workings in a way that allows students to clearly visualise the information they're submitting. The idea is that students should see in real time what the workings are they are creating and create an agreed semantic between the user and backend services.

By showing users what we think they mean in real-time as they type, we allow them to understand what we're going to interpret their answers (and workings) as when we receive them. Tryal UI is backed up by a new powerful API service (coming soon) that will be capable of taking the data input via Tryal UI, interpret it and then mark it.

Tryal Graph

Tryal Graph is built on top of the graphical technology stack we use internally for the Tryal core. It allows students to manipulate and draw graphical elements. We then translate this back into a set of datapoints that can be used to assess whether a student has drawn the desired answer
